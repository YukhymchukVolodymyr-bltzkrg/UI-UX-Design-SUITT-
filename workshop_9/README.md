# ПЗ-9 Функція Auto layout у Figma. Створення картки товару

## Тема заняття  
Вивчення та практичне застосування функції **Auto layout** у Figma під час створення **картки товару (послуги)**. Опрацювання принципів побудови гнучких інтерфейсних компонентів, роботи з відступами, вирівнюванням, вкладеними авто-лейаутами, а також використання зображень та іконок.

---

## Хід роботи  

### 1. Опрацювання теоретичного матеріалу  
Я опрацював матеріали по темі **Auto layout у Figma**, де розглядаються:

- принципи побудови гнучких блоків інтерфейсу за допомогою **Auto layout**;  
- вирівнювання елементів усередині контейнера (по вертикалі та горизонталі);  
- налаштування **відступів (Padding)**, **відстані між елементами (Spacing)**, **закруглення кутів**;  
- вкладені авто-лейаути для створення складних компонентів (кнопки, картки, списки товарів).

### 2. Виконання практичного завдання  
Метою практичної роботи було **створити картку товару** з використанням **Auto layout**. Як приклад було обрано смартфон **Apple iPhone 17 Pro Max 256Gb Cosmic Orange**.

Основні кроки виконання:

- **Створення базового фрейму картки** з м’яким фоном та закругленими кутами.  
- Налаштування для фрейму **Auto layout (vertical)** із внутрішніми відступами та рівномірним розподілом блоків.   
- Побудова блоку **іконок дій** (порівняння, збереження, додаткові опції) у правій верхній частині картки за допомогою **вертикального Auto layout** з однаковими відступами між круглими кнопками.  
- Створення текстового блоку з назвою товару:  
  - «Смартфон Apple iPhone 17 Pro Max 256Gb Cosmic Orange».  
- Додавання блоку цін:  
  - стара ціна — сірим кольором та з перекресленням;  
  - акційна ціна **«69 999 ₴»** виділена великим червоним шрифтом для акценту;  
  - блок цін оформлений як окремий **горизонтальний Auto layout**.  
- Розміщення коротких **характеристик товару** у вигляді текстового списку під ціною (діагональ дисплея, тип матриці, об’єм пам’яті, камера, акумулятор тощо).  
- Додавання кнопки **«кошика»** у вигляді круглої іконки (імпортована через плагін іконок) з вирівнюванням по правому краю за допомогою Auto layout.

У процесі роботи були використані такі інструменти та прийоми Figma:

- **Frame**, **Rectangle**, **Ellipse** — базові елементи інтерфейсу;  
- **Auto layout** для побудови:  
  - основного контейнера картки;  
  - вертикального блоку контенту (зображення, текст, ціна, опис);  
  - горизонтального блока ціни та кнопки «кошика»;  
- налаштування **Spacing between items**, **Padding**, **Alignment**;  
- робота з **тінями та закругленнями кутів** для створення відчуття «картки»;  

---

### 3. Результат  
У результаті було створено **адаптивну картку товару смартфона** з використанням Auto layout. Картка містить:

- зображення товару;  
- назву та короткий опис;  
- стару та акційну ціну з візуальним контрастом;  
- піктограми основних дій;  
- перелік ключових характеристик;  
- кнопку додавання до кошика.

Компонент побудований таким чином, що при зміні тексту, ціни або розмірів елементів **структура картки зберігається** завдяки налаштуванням Auto layout.

- закріпив навички роботи з **Auto layout** у Figma;
